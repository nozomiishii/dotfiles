- Activate the license
- General
  - Hotkeys:
    Pick Color: ⌃ + ⌥ + C
  - Clipboard Format >
    Check "Choose from Colors & Formats after picking"
- Appearance
  - Magnification >
    15x
- Code Style
  - Hex >
    check "Uppercase"
  - CSS Hex >
    check "Uppercase"
