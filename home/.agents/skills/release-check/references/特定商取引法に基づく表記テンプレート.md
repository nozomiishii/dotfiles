# 特定商取引法に基づく表記

| 項目 | 内容 |
| --- | --- |
| 販売事業者 | 【運営者氏名または法人名】 |
| 運営責任者 | 【運営者氏名】 |
| 所在地 | 請求があった場合、遅滞なく開示します。開示の請求は下記メールアドレスまでご連絡ください。 |
| 電話番号 | 請求があった場合、遅滞なく開示します。開示の請求は下記メールアドレスまでご連絡ください。 |
| メールアドレス | 【問い合わせ先メールアドレス】 |
| 販売価格 | 【プラン名】: 月額【◯◯◯】円(税込)または年額【◯◯◯◯】円(税込)。無料プランで利用できる機能の範囲はサービス内の表示をご確認ください。 |
| 販売価格以外に必要な料金 | インターネット接続にかかる通信料(ユーザーのご負担となります) |
| 支払方法 | クレジットカード決済(【決済代行。例: Stripe】) |
| 支払時期 | 申込み時に初回の料金が課金されます。以降は、解約手続きが行われない限り、契約期間の満了ごとに自動更新され、更新日に課金されます。 |
| サービスの提供時期 | 決済完了後、直ちにご利用いただけます。 |
| 契約期間・自動更新 | 月額プランは1か月ごと、年額プランは1年ごとに自動更新されます。 |
| 解約(キャンセル) | 【解約導線。例: サービス内の導線から遷移できる Stripe カスタマーポータル】にて、いつでも解約手続きができます。次回更新日の前までに解約手続きを行うと、以降の課金は発生しません。解約後も支払済みの契約期間の満了まではご利用いただけます。 |
| 返品・返金 | デジタルサービスの性質上、決済完了後の返金には応じられません。契約期間の途中で解約した場合の日割り返金も行いません。アカウントを削除した場合、契約は即時に終了し、残期間分の返金は行いません。 |
| 動作環境 | インターネット接続環境および最新版の主要ブラウザ(【動作確認済みブラウザを列挙】)。【ハードウェア要件があれば記載。例: 録音機能の利用にはマイクが必要です】 |
| 特別条件 | サービスの内容・提供条件は、利用規約に基づき変更されることがあります。 |
