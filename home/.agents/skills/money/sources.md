# 特典・割引の一覧サイト

/money の「安い道の探索」で参照するカタログ。手入れの仕方は [SKILL.md](SKILL.md) の「一覧サイトの手入れ」。

- [GitHub Student Developer Pack](https://education.github.com/pack): 学生向け開発ツール特典の一覧
- [Dev Starter Pack](https://devstarterpack.io/): スタートアップ・個人開発者向けの割引・特典の一覧
- [free-for-dev](https://github.com/ripienaar/free-for-dev): 開発者向けサービス無料枠の一覧。Web 版の free-for.dev は JS 描画でエージェントから読みにくいため repo を参照する
- [StartupPerks](https://startup-perks.com/): スタートアップ・founder 向けのインフラ・SaaS 特典の一覧。登録不要で閲覧できる
- [freestuff.dev](https://freestuff.dev/): 開発者向け無料サービス・特典の一覧
- [ミラサポplus](https://mirasapo-plus.go.jp/): 中小企業庁の補助金・助成金ポータル。補助金検索と公募スケジュール
